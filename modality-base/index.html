About modality-base

Layer that contains modules for the technical foundation of Modality.

Project Modules

This project has declared the following modules:

Name Description
modality-base-backoffice-activities-all Convenient module that embeds all back-office activities modules of the Base layer.
modality-base-backoffice-activities-filters The back-office Filters activity to create, modify, delete filters (displayed in other activities).
modality-base-backoffice-activities-home The back-office Home page that displays activities as browsable tiles (only the authorized ones are displayed).
modality-base-backoffice-activities-operations The back-office Operations activity.
modality-base-backoffice-masterslave Some class materials to build master/slaves views, with optionally an additional group view.
modality-base-backoffice-operations-filters Defines the basic operations that a back-office user can execute on filters.
modality-base-backoffice-operations-generic Defines some generic operations that a back-office user can execute on any entity (ex: copy to clipboard, set a field, toggle a boolean field).
modality-base-backoffice-operations-snapshot Defines operations to create a snapshot of an entity.
modality-base-client-activity Define some generic class materials to build Modality client activities.
modality-base-client-activity-organizationdependent Define some class materials to build organization-dependent client activities.
modality-base-client-application Define a generic Modality JavaFX application (common code to the back-office and front-office).
modality-base-client-busconfig Module that defines the default client bus configuration (with configuration variables and default values).
modality-base-client-css Cross-platform interface module for CSS (empty).
modality-base-client-css-java Java implementation of the CSS module (using JavaFX CSS).
modality-base-client-css-web Web implementation of the CSS module (using HTML CSS).
modality-base-client-entities Utility classes for working with entities on client side.
modality-base-client-gantt-canvas A zoomable Gantt canvas that can displays years, months, weeks and days over a time window.
modality-base-client-gantt-fx A set of Gantt-related JavaFX properties (time window, selection, visibility, etc...).
modality-base-client-i18n The Modality implementation of i18n with English & French dictionaries.
modality-base-client-icons A resource module containing most icons used in Modality.
modality-base-client-navigationarrows-java Plugin module to declare the backward and forward navigation operations in Java.
modality-base-client-operationactionsloading Module responsible for loading all information about operations from the database on client start.
modality-base-client-presentationmodel Set of HasXXX JavaFX markers designed for building presentation models.
modality-base-client-time-theme A theme for visual component representing time.
modality-base-client-util A set of client-side utility classes (translate function & route interpolator).
modality-base-client-validation Modality implementation of client validation support.
modality-base-frontoffice-activities-account The front-office Account activity.
modality-base-frontoffice-activities-alerts The front-office Alerts activity.
modality-base-frontoffice-utility A set of utility classes for the front-office.
modality-base-server-application-all Convenient module that embeds all server modules of the Base layer.
modality-base-server-application-min Convenient module that embeds the minimal set of server plugin modules required for the Base layer.
modality-base-server-datasource Module that defines the database connection (configurable variables with default values), and test the connection on server start.
modality-base-shared-domainmodel Module that loads the domain model in memory (on both client & server).
modality-base-shared-entities Module that defines a set of Java wrappers for domain entities (easier to interact with Java code).
