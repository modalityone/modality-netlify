About kbs3

Kadampa Booking System (3rd generation) built on top of Modality & WebFX.

Project Modules

This project has declared the following modules:

Name Description
kbs-backoffice-application The KBS Back-Office application (cross-platform non-executable module).
KbsBackoffice The Gluon executable module of the KBS Back-Office (for tablets).
kbs-backoffice-application-gwt The GWT executable module of the KBS Back-Office (for browsers).
kbs-backoffice-application-openjfx The OpenJFX executable module of the KBS Back-Office (for desktops & development).
kbs-frontoffice-activities-home The customized home page of the KBS Front-Office that displays the Kadampa news & podcasts.
kbs-frontoffice-application The KBS Front-Office application (cross-platform non-executable module).
KbsFrontoffice The Gluon executable module of the KBS Front-Office (for mobiles & tablets).
kbs-frontoffice-application-gwt The GWT executable module of the KBS Front-Office (for browsers).
kbs-frontoffice-application-openjfx The OpenJFX executable module of the KBS Front-Office (for development).
kbs-server-application The KBS Server application (cross-platform non-executable module).
kbs-server-application-vertx The Vert.x executable module of the KBS Server.
kbs-server-newsimport The server module that regularly imports the news from kadampa.org to the database.
kbs-server-podcastsimport The server module that regularly imports the podcasts from kadampa.org to the database.
